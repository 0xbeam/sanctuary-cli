How it works

Paste any public GitHub URL. No config, no setup, no dependencies.

Instant analysis
Routes, APIs, agents, components, database schema — detected in seconds.
03
Interactive dashboard
Six-tab dashboard. Searchable, exportable, shareable. Always current.
What it detects
Frameworks
Next.js, Nuxt, Remix, FastAPI +
Framework, version, runtime, and features like Tailwind, Supabase, Prisma, Zod.
Routes & APIs
App Router, tRPC, GraphQL
Every route and API endpoint with HTTP methods, auth detection, and schema types.
AI Agents
Models, tools & connections
Model name, temperature, system prompt, tools used, and agent-to-agent calls.
Components
UI & feature split
Base UI components separated from feature-level compositions.
Database
Tables & migrations
Prisma schemas and SQL migrations mapped to your data model.
Languages
TypeScript, Python, Go, Rust
Multi-language support with framework detection for each ecosystem.
